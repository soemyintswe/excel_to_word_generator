# Excel to Word Generator

ဤ Tool သည် Excel ဖိုင်အတွင်းရှိ Data များအား Word Document များသို့ အလိုအလျောက် ပြောင်းလဲပေးသည့် Application ဖြစ်သည်။

## အသုံးပြုနည်း (How to Use)
၁။ **Environment အဆင်သင့်ဖြစ်စေရန်**: Terminal တွင် `pip install -r requirements.txt` ရိုက်၍ လိုအပ်သော Library များ သွင်းပါ။
၂။ **Template ပြင်ဆင်ခြင်း**: `templates/` folder ထဲတွင် မိမိအလိုရှိသော Word Template ဖိုင်ကို ထည့်သွင်းပါ။
၃။ **Data ပြင်ဆင်ခြင်း**: `data/` folder ထဲတွင် လိုအပ်သော Excel အချက်အလက်ဖိုင်များကို ထည့်သွင်းပါ။
၄။ **စတင်အသုံးပြုခြင်း**: `Run.bat` ဖိုင်ကို Double Click နှိပ်၍ သို့မဟုတ် `python main.py` ဖြင့် စတင်ပါ။
၅။ **ရလဒ်ထုတ်ယူခြင်း**: ထွက်လာသော Word ဖိုင်များကို `output/` folder ထဲတွင် ကြည့်ရှုနိုင်ပါသည်။

## လိုအပ်ချက်များ (Requirements)
- Python 3.x
- `requirements.txt` တွင်ပါဝင်သော Library များ
